# Memoria ― ヨルシカ世界観マップ（文学装丁版）

ヨルシカの楽曲を「まとめ」ではなく「空間」として歩くための世界観データベース。
[memoria-yorushika](https://github.com/shimi-works/memoria-yorushika) のデザイン全面改訂版です（データ・機能は同一）。

## デザインコンセプト ── 文学・夏・夜

- **頁は原稿用紙**: 生成りの紙にごく薄い方眼の罫。文字は墨、罫と印は弁柄の朱
- **星図と地図だけが夜**: 世界観マップは「頁に開いた藍の夜の窓」。蛍がひとつふたつ飛ぶ
- **金の箔押し・グロー発光は全廃**: 活版・製本の意匠（内題・朱印・奥付・年譜）に置き換え
- 「ユーザーの読み」の書き込み欄は本物の原稿用紙のマス目

## 構成

- 単一HTML（`index.html`）・ビルド不要・APIキー不要
- 地図のみ Leaflet + OpenStreetMap / CARTO light タイル（セピア調整で紙に馴染ませている）
- タブ: 世界観マップ／楽曲／物語／歩み／聖地アトラス／この地図について

## 方針

- 歌詞の全文掲載はしない（著作権）。原作・公式へのリンクで誘導する
- 考察は「公式／有力考察／ユーザーの読み」の3層を混ぜずに常時表示する
